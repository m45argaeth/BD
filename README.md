# Bias Detector

> Discover the hidden shortcuts your brain uses.

An educational playground that helps you identify cognitive biases through
interactive scenarios, a searchable bias library, and a daily challenge.

Part of the same series as **Everything Becomes Numbers**, **Token Explorer**,
and **Text To Binary** — sharing the same design system, header/footer, and
cross-links.

## Stack

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS + shadcn/ui-style components
- next-themes (light default + dark toggle), sonner, lucide-react
- Custom i18n (`id` / `en`, default `id`)
- No database — fully static scenario & bias dataset

## Develop

```bash
npm install
npm run dev
```

Deploy-ready for Vercel.

## Note

Cognitive biases are normal mental shortcuts. Everyone experiences them.
This tool is educational and **not** a psychological assessment.
